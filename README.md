# Vecno Wallet - a non-custodial wallet for Vecno blockDAG

## What is Vecno Wallet?

Vecno Wallet is a non-custodial wallet for Vecno, available for Android and iOS. It is written in [Dart](https://dart.dev) using [Flutter](https://flutter.dev).

| Link | Description |
| :----- | :------ |
[vecnofoundation.org](https://vecnofoundation.org) | Vecno Blockchain Homepage

## Contributing

* Fork the repository and clone it to your local machine
* Follow the instructions [here](https://flutter.io/docs/get-started/install) to install the Flutter SDK
* Setup [Android Studio](https://flutter.io/docs/development/tools/android-studio) or [Visual Studio Code](https://flutter.io/docs/development/tools/vs-code).

## Building

Android: 
```bash
flutter build apk
```

iOS: 
```bash
flutter build ios
```

If you have a connected device or emulator you can run the app right from your development machine.

Debug mode:
```bash
flutter run
``` 

Release mode:
```bash
flutter run --release
```
